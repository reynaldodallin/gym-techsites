## 🚀 Deploy no Cloudflare Pages

### Opção 1: Deploy Automático via Dashboard

1. Acesse: `https://dash.cloudflare.com/`
2. Login com: `reynaldodallin@gmail.com`
3. Vá em: **Workers & Pages** → **Create Application** → **Pages** → **Connect to Git**
4. Conecte este repositório: `reynaldodallin/gym-techsites`
5. Configurações de Build:
   - **Framework preset**: None (HTML estático)
   - **Build command**: (deixe vazio)
   - **Build output directory**: `/`
6. Clique em **Save and Deploy**

### Opção 2: Deploy via Wrangler CLI

```bash
npm install -g wrangler
wrangler login
wrangler pages project create fitpower-gym
wrangler pages deploy . --project-name=fitpower-gym
```
